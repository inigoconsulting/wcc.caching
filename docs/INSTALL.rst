wcc.caching Installation
------------------------

To install wcc.caching using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.caching`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.caching

* Re-run buildout, e.g. with:

    $ ./bin/buildout
